# Goldener Apfel – Website

Website der Traditions-Wirtschaft **Goldener Apfel** in Rottweil.
Statische Seite (HTML/CSS), ohne Build-Schritt – einfach hosten.

## Live schalten mit GitHub Pages
1. Dieses Repo öffnen → **Settings** → **Pages**
2. Unter **Build and deployment → Source**: „**Deploy from a branch**"
3. Branch: **main**, Ordner: **/(root)** → **Save**
4. Nach ~1 Minute ist die Seite erreichbar unter
   `https://SavioLD.github.io/goldener-apfel-website/`

## Eigene Domain (goldenerapfel-rottweil.de)
- In **Settings → Pages → Custom domain** die Domain eintragen.
- Beim Domain-Anbieter einen `CNAME`-Eintrag `www` → `savioLD.github.io`
  bzw. die vier A-Records von GitHub Pages für die Root-Domain setzen.

## Bilder ergänzen
Fotos mit den in `assets/BILDER-README.txt` genannten Dateinamen in den
Ordner `assets/` legen. Fehlt ein Bild, zeigt die Seite automatisch einen
dezenten Platzhalter.
