# Goldener Apfel – Website (komplettes Paket)

Alle Dateien liegen im Root (flache Struktur), direkt für GitHub Pages.

## Hochladen
GitHub-Repo -> Add file -> Upload files -> ALLE Dateien aus diesem Ordner
hineinziehen (inkl. der .jpg-Bilder und .nojekyll) -> Commit changes.
Wichtig: index.html muss im Root liegen.

## Live schalten
Settings -> Pages -> Deploy from a branch -> main / (root) -> Save.

## Noch eintragen
Telefon / WhatsApp / Anschrift (aktuell Platzhalter) in index.html.
